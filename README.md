# YunBook
基于微信的云书架

# 功能说明
通过微信扫描图书的isbn编码，获取图书信息，存入数据库，调用相关的图书信息展示

# 依赖
Bmob后端云 

微信企业号

豆瓣读书api

# 配置

根据微信企业号开发文档和Bmob后端相关文档进行配置，主要用到的是应用key和应用密钥

配置config.php文件即可
